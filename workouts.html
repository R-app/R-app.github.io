Workout
Run 1 Mile
100 Pullups
200 Pushups
300 Situps
Run 1 Mile
-Murph
This is a tough workout so take your time. A slightly easier version involves doing the strength portion of the workout out of order- this means treat each set figure (i.e. 100 pullups) as a number to achieve, but without regard for the order in which it is achieved. For example, try doing 20 sets of 5 pullups, 10 pushups, and 15 situps.
